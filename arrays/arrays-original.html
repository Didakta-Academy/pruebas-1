Array

Un array es un tipo de variable que dentro puede tener varios valores

let myWeapons = ["Espada", "Escudo", "Arco"];

console.log(myWeapons);

Como ves en el array puede contener la cantidad de elementos que tú quieras y pueden ser tanto números como texto. Incluso puedes mezclarlos si quieres

let myWeapons = ["Espada", "Escudo", "Arco"];

let favNumbers = [3, 7, 25, 57, 100];

let mixedArray = ["Lluvia", 400, "Sol", "Nublado", 37];

Como has visto en el primer ejemplo si pones el array en un console.log sale todo entero, pero igual quieres un solo elemento. Esto puede hacerse con esta sintaxis

arrayName[numero del elemento];

pero tienes que saber que el primer elemento empieza por el número 0

let myWeapons = ["Espada", "Escudo", "Arco"];

let myWeapons = ["Espada", "Escudo", "Arco"];
console.log(myWeapons[0]);

Consola: Espada

Lo mismo que puedes usar un elemento para mostrarlo por consola puedes usarlo para otras circunstancias como ponerlo en un if o en una variable


let myWeapons = ["Espada", "Escudo", "Arco"];

let favWeapon = myWeapons[0];

console.log(favWeapon);

Consola: Espada

Con el índice del elemento también puedes cambiar su valor en la posición del array


let myWeapons = ["Espada", "Escudo", "Arco"];

console.log(myWeapons[0]);

myWeapons[0] = "Hammer";

console.log(myWeapons[0]);

Consola:
Espada
Hammer

El navegador conoce una serie de instrucciones que puede usar con los arrays. Por ejemplo puede conocer la longitud de un array y decírtela con array.length;


let myWeapons = ["Espada", "Escudo", "Arco"];

console.log(myWeapons.length);

Consola: 3

El navegador también sabe cómo añadir nuevos elementos al final del array usando array.push(newElement)


let myWeapons = ["Espada", "Escudo", "Arco"];

console.log(myWeapons);

myWeapons.push("Hammer");

console.log(myWeapons);

O eliminar el último usando array.pop();


let myWeapons = ["Espada", "Escudo", "Arco"];

console.log(myWeapons);

myWeapons.push("Hammer");

console.log(myWeapons);

myWeapons.pop();

console.log(myWeapons);
